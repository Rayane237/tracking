# Notes design

L'interface s'inspire de l'image fournie :

- Bloc cargaison clair avec véhicule, quantité et couleur.
- Carte de suivi en temps réel avec ports, position et itinéraire.
- Timeline d'escales lisible, professionnelle et rassurante.
- Palette Dubai Global Express : rouge, noir, blanc, gris.
- Style premium : cartes propres, ombres discrètes, coins arrondis, micro-animations.
